What is DOM ? and what is the purpose of DOM ?

DOM means Document Object Model.The Document Object Model represents each page in a tree-like structure to make it easier to access and manage the elements of this page.It is not a programming language.It's just a root elements of html pages. The importance of DOM are significant. DOM allows us remove create or change any elements or tags from the website document.By DOM we can randomly access element of html pages.

How will you select the HTML elements of DOM ? and Name the DOM Methods

DOM means Document Object Model.By using DOM We can access any random elements of html pages.Using some specific selectors We can access HTML elements using DOM.The Selectors are

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()

What is Event Bubble ?

The Extension of an event in JS Dom element is called event bubble. The event triggers target element and the target element triggers the parent element until it triggers the outermost element.The flow of event from target element to outermost parent element is called event bubble.

What is Callback Function ? And why will you use it ?

If we call a function in another function it is called Callback Function.The main function takes another function as a parameter and calls it in the main function.It makes our coding easy and convinient
